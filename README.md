# MouseInfoLinux

Utilitário gráfico para capturar coordenadas e cores (RGB/HEX) em tempo real.

## Instalação
```bash
pip install mouseinfolinux
```

## Como usar
```bash
mouseinfolinux
ou
python3 -m mouseinfolinux
```
